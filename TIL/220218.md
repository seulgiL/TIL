## 2752 정렬

```python
my_list = list(map(int, input().split()))

for i in range(2, 0, -1):
    for j in range(i):
        if my_list[j] >= my_list[j+1]:
            my_list[j], my_list[j+1] = my_list[j+1], my_list[j]

print(*my_list)
```

- 간단하게 버블소트로 풀려고 하였으나, 버블 소트를 구현하는데 많이 헤매었다.
